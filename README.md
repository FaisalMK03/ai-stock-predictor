# 🇸🇦 Tadawul Stock Market Predictor

A production-style AI-powered web app that predicts Saudi stock prices using historical data from the Tadawul market.

Built using Machine Learning and deployed with Streamlit.

---

## 🚀 Features

- 📈 Predict next-day stock prices using Linear Regression  
- 📊 Interactive charts (Actual vs Predicted + Future Forecast)  
- 🧠 Feature engineering (MA10, MA50, Returns, Lag features)  
- 🇸🇦 Focus on Saudi companies (Aramco, STC, Al Rajhi, etc.)  
- 🎯 Forecast multiple future days (3–30 days)  
- 💡 Clean Bloomberg-style UI  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib  
- yFinance API
